# 🧹 SCT_DA_2 - Data Cleaning and Preparation

## 📌 Project Overview

This project was completed as part of the **SkillCraft Technology Data Analytics Internship**. The objective was to clean and prepare raw data using Python and Pandas to ensure data quality and readiness for analysis.

## 🎯 Objectives

* Identify and handle missing values
* Remove duplicate records
* Correct and convert data types
* Prepare a clean dataset for analysis
* Export cleaned data to a new CSV file

## 🛠 Tools & Libraries

* Python
* Pandas
* Jupyter Notebook / Google Colab

## 📂 Dataset

The dataset was loaded into a Python environment and processed using Pandas for data cleaning and preparation.

## 🔍 Data Cleaning Steps

* Checked dataset structure and data types
* Identified missing values
* Handled null values appropriately
* Removed duplicate rows
* Converted columns to appropriate data types
* Exported the cleaned dataset
